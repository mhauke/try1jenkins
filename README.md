# Your Python Project


This project calculates the factorial of a number and includes tests using `pytest`.

## Directory Structure
factorials/ 
├── Jenkinsfile 
├── README.md 
├── requirements.txt 
├── src/ 
│   ├── init.py 
│   └── factorial.py 
└── tests/ 
    ├── init.py
    └── test_factorial.py

## Usage

To calculate the factorial of a number, run:

```
sh python src/factorial.py <number>
```
